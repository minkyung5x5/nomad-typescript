### 빌드 방법

1. 깃헙 레포지토리 클론

   ```$ git clone https://github.com/minkyung5x5/nomad-typescript.git```
2. Terminal에서 패키지 설치
   
   ```$ npm install```
   
3. Terminal에서 실행
   
   ```$ npm run dev```
